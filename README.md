## Análise de Dados de Ações com Pandas e Matplotlib

Este projeto realiza uma análise de dados do mercado de ações utilizando as bibliotecas Pandas e Matplotlib no Python, combinando manipulação de dados e visualização gráfica para interpretar o comportamento dos ativos financeiros.

A análise inclui o tratamento dos dados, acompanhamento da variação dos preços das ações e criação de gráficos que auxiliam na identificação de tendências e padrões do mercado.

### Objetivos do projeto
- Analisar dados históricos de ações
- Manipular informações financeiras com Pandas
- Criar gráficos e visualizações com Matplotlib
- Identificar tendências e variações de preços
- Praticar análise de dados financeiros com Python

### Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
